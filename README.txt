O codigo javascript em 'index_1.html', ao ser executado, irá renderizar uma página em branco com um botão
CLICK.

Se o CLICK for precionado antes das 18:00, aparecerá duas mensagens:

1) tempo restante para o desligamento do computador. Formato-> (hh:mm:ss);
2) comando para desligar o computador às 12:00. shutdown -s -t segundos_restantes;
3) comando para desligar o computador às 18:00. shutdown -s -t segundos_restantes;
4) comando para o cancelamento do desligamento agendado
