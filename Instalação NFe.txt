 
Instruções para Instalação da NFe 2.0 para iniciar em 01/04/2011

OBS: as chaves estão no TXT anexado


1. se ainda não foi instalado, instalar o certificado digital

2. se ainda não foi instalado, instalar DOT Net 2.0 ( www..com.br/dotnetfx20.exe )

3. Instalar o módulo NFe ( www..com.br/instnfe3.exe )

4. Instalar Faturamento ( www..com.br/instwfat.exe )  e Estoque ( www..com.br/instest.exe )

5. Registrar a DLL NFe com os comandos abaixo:

C:...
	cd \
	cd nfe
	registraDLL_NFe_2G
	exit

6. se ainda não foram instaladas, instalar as fontes de código de barra no windows (painel de controle), as fontes estão na pasta C:\NFE\FONTES

7. Instalar a licença para transmissão de NFe (menu utilitários do faturamento). Verificar que são campos distintos para as versões 1 e 2

8. Definir no cadastro da empresa do módulo Faturamento o Código de Regime Tributário: 
	"1=Simples Nacional", "2=Simples Nacional - com excesso de sublimite da receita bruta", "3=Regime Normal"

9. Se a empresa ainda não fez, fazer credenciamento no ambiente de homologação da SEFAZ ( www.sefaz.pe.gov.br ) conforme o roteiro de credenciamento



REGISTRO NFE CARTA DE CORREÇÃO
Registradll_nfe_2g.bat
